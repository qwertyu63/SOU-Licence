Shared Open Universe Licence

The Shared Open Universe Licence is a public licence, allowing fan works to be created and shared freely and contributing to the Open Universe, a pool of settings which anyone can use, allowing fans and the original creator to build on each other's work.

Basic Terms:

If a work is released under the Shared Open Universe Licence, you are permitted to:
  • Create fan works derived from it and;
  • Share those fan works in any way you see fit, even commercially.
But, you are required to follow a couple rules:
  • You must give credit to the original work and its creator.
  • You must release your fan work under the Shared Open Universe Licence.
Unlike most similar public licences, you are not granted any rights over the original work itself.

This summary highlights only some of the key features and terms of the actual license. It is not a license and has no legal value.
You should carefully review all of the terms and conditions of the actual license before using the licensed material.
